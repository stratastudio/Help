Commandes

1. Ajouter nouvelle commande

Pour ajouter une nouvelle commande,

bolt accédez à la page Commandes dans le menu menu et cliquez sur (note_add Nouvelle commande) en haut à gauche:

Dans la page (note_add Nouvelle commande), au moins un produit doit être ajouté à la liste pour que la commande puisse être sauvegardée.

Un Bon Nº sera automatiquement ajouté suivant le dernier numéro enregistré. Il peut être modifié manuellement si nécessaire.

Etape 1 - Sélectionner client

Dans Polysoft, un client peut être de 2 types:

  • Un client individuel person qui passe une commande en son nom, ou
  • Une société domain

Pour les clients individuels, leur nom complet est affiché pour représenter leur compte. (Voir comment créer un client individuel pour plus de détails).

Pour les sociétés, c'est un peu différent:

Pour qu'une société soit sauvegardée, il faut une seule personne liée au compte de la société en tant que point de contact. (Voir comment créer une société pour plus de détails).

Dans la selection des clients, à la fois les clients individuelsperson et les sociétés domain seront affichés dans la même liste.

L'icône (person,domain) avant le nom indique s'il s'agit d'une société ou d'un client individuel.

boltCliquez sur le champ Client pour choisir un client dans la liste.

Les sociétés auront leur nom de Point de Contact affiché en jaune. Dans ce cas, la société domain ExampleTex

a son point de contact Abdessamad 001 en jaune

Etape 2 - Ajouter articles

La zone en tirets jaunes est la sélection de catégorie.

Pour afficher les articles sur la liste, il faut sélectionner la catégorie associée. Ici, la catégorie check Visline - 7g avec Colle (rose) est sélectionnée et le produit associé apparaît sur la liste

boltAjoutez un article et taper une quantité.

Dans cet example, 8003 est sélectionné avec une quantité de 12 unités. Le bouton Ajouter est activé pour ajouter la commande à la liste.

Dans le même example, on ajoute 1 unité de Fil canette et 3 unités de Visline - sans colle (vert)

Clients

Pour ajouter un nouveau person client,

bolt Allez à la page Clients dans le menu menu et sélectionnez (person_add Nouveau client) en haut à gauche:

boltAjoutez le Nom et Prénom du client

Dans la page person_add Nouveau client, il est recommandé d’entrer autant de détails que possible pour éviter les doublons de clients. Cependant, seul le Prénom est requis pour créer un nouveau client.

Lorsque vous entrez le Prénom uniquement dans le champs et cliquez sur Enregistrer, un message apparaîtra qu’un numéro, dans ce cas “002”, sera automatiquement généré comme Nom du client.

Ce client sera enregistré sous le nom Amine 002 car il y a déjà un Amine 001 dans la liste des clients.

C’est pourquoi il est préférable d’ajouter un Nom et un Prénom pour identifier le client.

boltAjoutez l'adresse de livraison du client dans le cas necessaire

Dans le cas ou vous devez ajouter une nouvelle adresse, cliquez sur add_circle dans le champ d’adresse

Vous serez redirigé vers la page import_contacts Nouvelle adresse. Entrez les informations nécessaires et cliquez sur Enregistrer.

Voir ajout et modification d’adresses pour plus d'information.
